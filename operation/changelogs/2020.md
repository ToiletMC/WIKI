# 2020年

## 7月12日

#### 新增内容

* 服务器更新至1.16.1！
* 增加对基岩版客户端支持！
  * 你必须满足一个条件：拥有JAVA正版，并且通过正版认证。
  * 在使用基岩版登陆时，会要求输入JAVA正版账号和密码，然后进行数据互通。

{% hint style="info" %}
基岩版服务器地址：be.toiletmc.net           端口：20245
{% endhint %}
